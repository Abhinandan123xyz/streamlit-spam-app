 SMS Spam Classification Web App

A Machine Learning web application that predicts whether a message is **spam** or **not spam** using NLP techniques.

Live Demo: https://abhi-spam-app.streamlit.app/

 Features
- Real-time SMS classification
- TF-IDF vectorizer & Logistic Regression model
- Text preprocessing: stopword removal, stemming, tokenization
- Fully deployed on Streamlit Cloud

 Tech Stack
- Python
- Scikit-Learn
- NLTK
- Streamlit
- Pandas / NumPy

 Model Performance
| Metric | Score |
|--------|-------|
| Accuracy | ~96% |
| Precision | High |
| Recall | High |
